# Brutal Reader

**Strip any webpage to pure readable text. One click.**

No ads. No sidebars. No popups. No subscribe banners. Just the article, on warm paper.

![Brutal Reader Demo](demo.gif)

---

## Install

### Chrome / Edge (manual sideload — free)

1. Download the latest release → **[brutal-reader.zip](https://github.com/YOUR_USERNAME/brutal-reader/releases/latest)**
2. Unzip it
3. Go to `chrome://extensions`
4. Enable **Developer Mode** (toggle, top right)
5. Click **Load unpacked** → select the unzipped `brutal-reader` folder
6. Pin it to your toolbar

### Firefox

Coming soon via addons.mozilla.org

## How it works

Click the extension icon on any article. The page strips down to:

- Clean serif typography on warm paper texture
- Article text only — buttons, ads, sidebars, share prompts all gone
- Inline images preserved
- Word count + estimated read time
- Scroll progress bar
- Press **ESC** or click **✕ Exit** to return to the original page

Works well on: Substack, news sites, blogs, Medium, and most article pages.

---

## Why

Web pages are built to extract attention, not deliver it. Brutal Reader flips that.
